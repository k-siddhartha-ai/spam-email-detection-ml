# spam-email-detection-ml
Spam email classification using Machine Learning in Python.
## Live Demo
https://huggingface.co/spaces/<your-username>/<your-space-name>

## How it works
This app checks whether a message is spam or not spam using machine learning.
If the model confidence is low, it returns an "UNCERTAIN" result instead of forcing a wrong prediction.

## Note
This is a demo project trained on limited data and is meant for learning purposes.
